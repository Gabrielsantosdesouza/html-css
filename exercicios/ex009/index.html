O poder das listas

Listas Ordenadas

  1. arroz
  2. feijão
  3. carne
  4. sabão

para criiar uma lista ordenada, basta usar a tag OL para delimitar a lista e LI para identificar cada item da lista

Para alterar a marcação de Numero para letra ou outros tipos, basta usar a tag TYPE juntamente com a tag OL

E para alterar a contagem para iniciar em um determinado numero ou letra, basta usar a tag START após a tag TYPE

Listas não ordenadas

  • maconha
  • cocaína
  • lolo
  • papel

Para criar uma lista desordenada basta usar a tag UL para delimitar a lista e a tag LI para criar cada um dos seus itens internos

Misturando as coisas

  1. farinha
  2. açucar
  3. ovo
    1. fermento
    2. agua
    3. trigo
  4. sal
  5. povilho
  6. chocolate

Para criar uma lista dentro de outra lista basta abrir a tag OL dentro de outra tag OL

  1. D
  2. g
  3. h
    1. t
    2. t
    3. f
  4. g
  5. d
  6. r

Note que os ítens da lista interna a b c estão deslocados para a direita em relação ao ítem 3 do qual essa sub lista é filha
